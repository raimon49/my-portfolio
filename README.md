## Debugging

Run local server

```sh
$ mkdocs serve -a <IP:PORT>
```

## Manage dependencies

Update dependencies

```sh
$ pip-compile -U requirements.in
```

## Deployment

```sh
$ mkdocs gh-deploy --clean
```
